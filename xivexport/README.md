# XIV Export

Data export tool for sending XIV content data to a searchable index.

# To Run

```cmd
python -m venv venv
# be sure to activate the environment 
pip install -r pyproject.toml
python -m __main__
```

Or with `uv` installed:

```cmd
uv venv
uv pip install -r pyproject.toml
uv run __main__.py
```
