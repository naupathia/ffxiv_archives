# FFXIV Archives

A searchable archive of text and lore content from Final Fantasy XIV.

## What is this?

Final Fantasy XIV contains thousands of lines of written content from quest dialogue to item descriptions, cutscene text, NPC dialogue, and much, much more. This project extracts as much text as possible from the game's data files and makes it searchable through a web interface.

Whether you're trying to remember a specific line of dialogue, researching lore for creative projects, or just want to browse through the game's writing, this tool lets you search across all of FFXIV's text content in one place.

## Features

- **Full-text search** across various types of game content
- **Filter by category** (quests, items, cutscenes, mounts, etc.)
- **Filter by expansion** (A Realm Reborn through Dawntrail)
- **Browse** through archived content without needing the game installed

## Components

- **Data Export Tool** - Extracts text content from FFXIV game data and stores it in a database
- **Web Search App** - A web interface for searching and browsing the archived content

## Data Sources

The archive includes text from:
- Main scenario and side quests
- Cutscenes
- Item descriptions
- Mount and minion descriptions
- Triple Triad card descriptions
- FATE descriptions
- Fishing log entries
- Status effects
- Random NPC dialogue
- And more
